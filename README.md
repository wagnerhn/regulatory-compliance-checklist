# Regulatory Compliance Checklist

This project provides a simple command-line tool to assess regulatory compliance practices based on key governance and internal control questions.

## Purpose

The tool simulates a basic compliance assessment process often used in Governance, Risk, and Compliance (GRC) environments.

## How it works

The script asks a series of compliance-related questions and calculates a compliance score based on the responses.

## Example

Is segregation of duties implemented? (y/n)

Is regulatory monitoring performed regularly? (y/n)

Compliance Score: 83%
Risk Level: Low
